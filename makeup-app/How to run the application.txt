To run this application 

  - Deploy this web application on any application server.

  - Create the database and tables in mysql, using the queries in the text file "Database queries". 

  - add database username, password in (DataSource.config, organization.properties)

  - Maven clean, install
  - run web main
  - keep mysql running

  - Access this application using url: http://localhost:8080/login

  - sign-up with your own username and password and login.

  - to get all appointments- use admin, admin
