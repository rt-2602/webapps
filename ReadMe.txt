Makeup application

This is a web application developed using springboot, spring MVC, thymeleaf, mysql, maven.

Deploy this web application on any application server.
Access this application usin url: http://localhost:8080/login

Use username and password: Jedu, Jedu to login or sign-up with your own username and password.

This application is also Multi-Lingual, you can access the application in 2 languages i.e Hindi, Spanish and English

For using application in hindi use url: http://localhost:8080/login?locale=hi

For using application in Spanish use url:  http://localhost:8080/login?locale=es
